## 📋 Configuración Inicial

Antes de comenzar, configura tu identidad en Git:

```bash
git config user.name "Tu Nombre"
git config user.email "tu-correo@ejemplo.com"
```

## 🔧 Comandos Esenciales de Git

### 📥 Obtener Cambios

| Comando           | Descripción                                              |
| ----------------- | -------------------------------------------------------- |
| `git clone <URL>` | Descarga el repositorio por primera vez                  |
| `git fetch --all` | Busca ramas y cambios nuevos en GitHub                   |
| `git pull`        | Descarga y fusiona los últimos cambios en tu rama actual |

### 🌿 Gestionar Ramas

| Comando                        | Descripción                                 |
| ------------------------------ | ------------------------------------------- |
| `git branch -a`                | Lista todas las ramas (locales y de GitHub) |
| `git checkout <rama>`          | Cambia a una rama existente                 |
| `git checkout -b <nueva-rama>` | Crea una rama nueva y te mueve a ella       |

### 💾 Guardar Cambios

| Comando                               | Descripción                            |
| ------------------------------------- | -------------------------------------- |
| `git add .`                           | Prepara todos los archivos modificados |
| `git commit -m "Mensaje descriptivo"` | Guarda tus cambios localmente          |
| `git push`                            | Sube tus cambios guardados a GitHub    |

---

## 📌 Tips Útiles

- ✅ Siempre usa mensajes de commit descriptivos
- ✅ Haz pull antes de empezar a trabajar
- ✅ Crea ramas para nuevas funcionalidades
- ✅ Revisa el estado con `git status` frecuentemente
